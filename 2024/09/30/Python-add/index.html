Python 的动态类型

  • 列表 (list): 存储任意类型的数据，长度可以动态变化
  • 元组 (tuple): 不可变列表，一旦创建，就不能修改其中的元素，存储固定的一组数据。
  • 字典 (dictionary): 无序的键值对集合，使用哈希表实现，查找速度非常快，键可以是任意不可变类型。
  • 集合 (set): 无序、不重复元素集合。用于快速判断元素是否在集合中，以及进行集合运算（交集、并集、差集等）

切片和列表

  • start：起始索引，默认为 0。
  • end：结束索引，不包括该索引对应的元素。
  • step：步长，默认为 1。正数表示从左向右，负数表示从右向左。

元组

不同数据用逗号分隔，用这种字面量的方式即可以创建一个元组，括号并不是必须的。

字典

字典类型也有一些限制：

  • 字典的 key 不可以是可变数据，也不能包含可变数据
  • 一个 key 只能对应一个 value
