正则表达式基本符号分类

字符类型

  • \d : 匹配数字 (0-9)
  • \D : 匹配非数字
  • \w : 匹配字母、数字或下划线
  • \W : 匹配非字母、数字或下划线
  • \s : 匹配空白字符 (空格、制表符等)
  • \S : 匹配非空白字符
  • . : 匹配任意字符 (除换行符)

位置类型

  • \b : 匹配单词边界
  • \B : 匹配非单词边界
  • ^ : 匹配字符串的开头
  • $ : 匹配字符串的结尾

数量限定符

  • + : 匹配前一个字符一次或多次
  • * : 匹配前一个字符零次或多次
  • ? : 匹配前一个字符零次或一次
  • {n,m} : 匹配前一个字符至少 n 次，至多 m 次
  • {n,} : 匹配前一个字符至少 n 次
  • {n} : 匹配前一个字符 n 次

修饰符

  • i : 忽略大小写
  • g : 全局匹配
  • m : 多行模式

字符类型例

正则：\d+
匹配文本：abc123def456
答案：123, 456

正则：\D+
匹配文本：123abc456def
答案：abc, def

正则：\w+
匹配文本：hello_world! 123
答案：hello_world, 123

正则：\W+
匹配文本：abc@#def$%ghi
答案：@#, $%

正则：\s+
匹配文本：hello world
答案： (3个空格)

正则：\S+
匹配文本：a b c
答案：a, b, c

正则：.
匹配文本：a1!b2@c3#
答案：a, 1, !, b, 2, @, c, 3, #
